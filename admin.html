Admin

Paste the admin token. It's printed in the terminal where you ran npm start, and stored in .admin-token.

This gate exists so a stray browser tab can't rewrite your city files. It is not a real authentication system, and this server should only ever listen on localhost.

    Identity

    Theme

    Five colours drive the whole page. The first four are used as text, so keep them dark enough to read on warm paper. Wash is the page background and should stay very pale.

    Hero

    Skyline

    Inner SVG markup on a 0 0 1200 240 canvas, ground at y=240. Group shapes into <g class="sky-far">, sky-mid and sky-near for depth. Leave blank to use a generic rooftop silhouette.

    Content
